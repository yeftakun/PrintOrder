# Installer prerequisites

Put the SumatraPDF installer here as:

```text
installer\SumatraPDF-Installer.exe
```

`PrintOrder.iss` will include it automatically when the file exists, detect whether SumatraPDF is already installed, and offer the prerequisite task only when needed.

Use the official SumatraPDF installer from:

```text
https://www.sumatrapdfreader.org/download-free-pdf-viewer
```

If SumatraPDF is bundled, keep the SumatraPDF license/source notice available with the release package.
